## 简介

后台UI模板 ELEMENT-UI

演示地址：https://micefind.com/element

## 技术栈

Vue3、TypeScript、Vite、ElementPlus

ElementPlus地址：https://element-plus.org/zh-CN/component/overview.html

## 基础功能

 - 明暗主题切换
 - 路由标签栏
 - 自定义系统主题色
